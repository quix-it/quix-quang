Utilizzo

Per fare il render delle traduzioni potete usare la relativa pipe transloco. Le chiavi sono le chiavi piatte del json che inserite i18n si consiglia di utilizzare la seguente logica nel crearle: modulo.pagina.entita.dettaglio

File .json

Nella directory asset create una directory i18n, al su interno aggiungete i file json delle lingue (es. en.json, it.json). Sempre in questa directory potete aggiungere le sotto-directory per gestire il lazy loading delle traduzioni, vi basta creare una directory con il nome definito nel modulo e al suo interno aggiungere i file delle lingue come per la root di i18n
